Anomaly Detector

Description

Anomaly_Detector is a modern web application that leverages AI to analyze log files and detect anomalies in real-time, making complex system monitoring accessible to everyone. Built with React 18 and TypeScript, the platform is powered by Cisco's Foundation-Sec-8B model - an 8-billion parameter AI specifically trained on enterprise network data and security logs.

This foundation model understands network infrastructure patterns and can identify subtle anomalies that indicate security breaches, system failures, or performance issues, The application integrates Hugging Face Transformers to translate technical findings into human-readable explanations, essentially converting cryptic error codes into actionable insights that anyone can understand.

The platform supports multiple file formats including JSON, CSV, XML, Syslog, and plain text, processing them in real-time with sub-second results. Users simply drag and drop log files into the interface, where Cisco's AI analyzes the data for unusual patterns while Hugging Face models generate user-friendly explanations. All processing happens securely in memory without data storage.

Technologies

React + Typescript, Cisco's Foundation-Sec-8B model
