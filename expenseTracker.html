Personal Accountant

Description

Recently, I explored n8n, a workflow automation platform that has been gaining attention in the Agentic AI space. I was impressed by how intuitive it felt, and within just a few hours I was able to build a fully functional AI-powered personal expense tracker. The setup works through a Telegram Messenger bot where I can simply text my daily expenses. Each entry is then saved directly into Google Sheets along with the date, type of expense, amount, and an optional description.

 

Configuring the Google Sheets and Drive APIs through OAuth2 credentials took some time, but once integrated, the system began updating expenses seamlessly. I also experimented with prompt engineering to refine the agentâ€™s responses. Initially, handling JSON outputs was challenging, but after iteration, I managed to ensure the agent could return natural language outputs while also asking relevant follow-up questions. One of the highlights was when I entered an expense in INR, and the agent automatically converted it into USD using the current exchange rate. While my daily expenses are minimal, having my own personal accountant running inside Telegram felt rewarding, and it opened my mind to the wide range of tasks I could automate next with AI agents.

Technologies

n8n, Google Cloud APIs(sheets, drive), Telegram Bot
