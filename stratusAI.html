StratusAI

Description

I developed StratusAI, a real-time network threat detection tool that processes raw packet traffic to generate live insights and anomaly alerts. The system leverages deep packet inspection with JA3 TLS fingerprinting to flag suspicious fingerprints and incorporates an autoencoder-based neural network for anomaly detection, achieving over 99% accuracy, precision, and recall. By aggregating traffic flows instead of individual packets, it reduces noise and provides richer context to identify scanning, beaconing, and data exfiltration behaviors.

 

To make these insights accessible, I built a full-stack React dashboard featuring real-time charts for bandwidth and anomaly levels, live alerts with detailed drill-downs, and a 3D globe visualization that maps source-to-destination IP traffic by threat level. The project was powered by Python (Scapy, Pandas, Torch, Watchdog), PyTorch for neural networks, JA3 for TLS fingerprinting, and custom flow aggregation logic.

This project is still in progress as I continue refining detection models and expanding dashboard features.

Technologies

Python (Scapy, Pandas, Torch, Watchdog), PyTorch for neural networks, JA3 for TLS fingerprinting
